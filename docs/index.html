How to Convert to Word (.docx)

  1. Open any document file (e.g., 01_System_Proposal.html) in Google Chrome or Microsoft Edge.
  2. Press Ctrl + P to open the print dialog.
  3. Select "Save as PDF" to get a clean PDF version, OR
  4. Open the .html file directly in Microsoft Word (File → Open → select the .html file) — Word will import it with formatting intact.
  5. Save as .docx from within Word.
