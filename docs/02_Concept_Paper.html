1. Problem

"Decentralized athlete training and nutrition tracking."

Athlete data is scattered across manual logs, personal devices, and isolated communication channels between athletes, coaches, and nutritionists. This fragmentation prevents the creation of a unified, reliable dataset that can accurately inform injury risk prediction, performance monitoring, and nutrition guidance. Without a centralized system, athletes often provide inconsistent or incomplete information, while support staff lack integrated access to the data they need to make timely, evidence-based decisions.

2. Proposed Solution

AfyaNexus is a centralized digital platform that unifies athlete training, recovery, and nutrition data. The name is derived from the Kiswahili word Afya (health) and the English word Nexus (connection), reflecting the system's purpose of connecting health data across all stakeholders in an athlete's support ecosystem.

The platform integrates manual athlete inputs, wearable device data from the T70 smartwatch, and a machine learning model to generate injury risk predictions. Role-based dashboards ensure that each user — athlete, coach, or nutritionist — accesses only the data relevant to their role, maintaining privacy and data integrity.

3. Key Features

🧍 Athlete Dashboard

Training logs, recovery records (sleep, soreness, mood), wearable sync, and injury risk score display.

🧑‍🏫 Coach Dashboard

Sponsored athlete performance overview, injury risk alerts, training trend analysis, and guidance tools.

🥗 Nutritionist Dashboard

Registered athlete nutrition logs, personalized diet plan creation, and performance-nutrition correlation charts.

⌚ Wearable Integration

Bluetooth sync with T70 smartwatch for automated capture of heart rate, sleep duration, and activity data.

🤖 ML Injury Prediction

Machine learning model processes training and recovery datasets to output an injury risk score and risk level.

🔐 Role-Based Access Control

Secure registration and authentication ensures each user only accesses data they are authorized to view.

4. Target Users

User Role Primary Function Data Access
Athlete Input and view personal performance data Own dashboard only
Coach Monitor athletes they sponsor and provide training guidance Sponsored athletes' dashboards only
Nutritionist Track registered athletes and deliver personalized diet plans Registered athletes' dashboards only

5. Innovation

AfyaNexus stands out through three core innovations:

  • Kiswahili-English Branding: The name AfyaNexus blends African identity with global technical language, making the system culturally relevant and professionally distinctive.
  • Wearable Device Integration: By syncing with the T70 smartwatch via Bluetooth, AfyaNexus reduces reliance on manual data entry, minimizing bias and improving the accuracy of ML predictions.
  • Machine Learning-Driven Predictions: The system goes beyond simple data logging by applying ML to generate actionable injury risk scores, enabling proactive rather than reactive athlete management.
